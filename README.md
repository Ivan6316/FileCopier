## Описание
**FileCopier** - это демонстрационная программа, показывающая возможности работы с файловой системой Windows API. Программа выполняет поиск документов определенных форматов в системных папках пользователя (Мои документы, Рабочий стол) и копирует их в новую директорию, названную именем текущего пользователя.

## Функциональность
1. Поиск файлов с заданными расширениями в директориях:
    1. Мои документы
    2. Рабочий стол
2. Поддерживаемые форматы файлов:
    1. Документы: .doc, .docx, .xls, .xlsx, .pdf, .rtf, .odt, .txt
    2. Изображения: .jpg, .jpeg, .png, .gif, .webp
3. Создание директории с именем текущего пользователя
4. Копирование найденных файлов в созданную директорию

## Этический аспект
